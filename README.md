# 🐼 판다마켓 (PandaMarket)

> 부트캠프 중 혼자 진행한 개인 프로젝트입니다.  
> 중고 거래 플랫폼을 클론하며 사용자 인증, 게시판 기능, 상품 등록 등 다양한 기능을 구현했습니다.

---

## 🖥️ 데모

👉 **배포 링크**: _곧 업데이트 예정입니다!_

---

## 📌 주요 기능

- 🔐 회원가입 및 로그인 / 로그아웃 기능
- 🛒 상품 등록 및 전체 상품 리스트 확인 가능
- 🗨️ 자유게시판 게시글 등록 및 리스트 확인 가능
- 🙋‍♀️ 마이페이지 (추후 추가 예정)
